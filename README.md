# qr-code
Generates QR codes for Haj pilgrims, enabling quick tracking and rescue if missing during pilgrimage

SOS QR Code for Haj Pilgrims
Generates QR codes for Haj pilgrims to enable quick tracking and rescue if missing during pilgrimage. (89 chars)
​

Features
- Unique QR code generation with pilgrim details
- Emergency SOS activation via scan
- Contact info & location data embedded
- Simple web/mobile interface
- Offline scannable codes

Quick Start
bash
git clone https://github.com/muzaffarsaiyed/qr-code.git
cd qr-code
npm install  # or pip install -r requirements.txt
npm start    # or python app.py

Usage
- Enter pilgrim details (Cover Number)
- Generate QR code
- Print/sticker on pilgrim ID badge
- Scan in emergency for instant location/contact info

Tech Stack
- Frontend: HTML/CSS/JavaScript/ASP.Net MVC
- QR Generation: qrcode library
- Database: SQL Server (See "SQL Query 1.txt" and "SQL Query 2.txt")
